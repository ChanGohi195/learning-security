# ls

ファイル・ディレクトリの一覧を表示。

## 基本

```bash
$ ls              # 一覧
$ ls -l           # 詳細表示
$ ls -la          # 隠しファイル含む詳細
$ ls -lah         # 人間が読みやすいサイズ表示
```

## 出力の読み方

```
-rwxr-xr-- 1 user group 5678 Jan 1 12:00 script.sh
│└┬┘└┬┘└┬┘ │  │    │     │        │        └── ファイル名
│ │  │  │  │  │    │     │        └── 更新日時
│ │  │  │  │  │    │     └── サイズ
│ │  │  │  │  │    └── グループ
│ │  │  │  │  └── 所有者
│ │  │  │  └── ハードリンク数
│ │  │  └── other権限
│ │  └── group権限
│ └── owner権限
└── ファイル種別（- = ファイル、d = ディレクトリ、l = シンボリックリンク）
```

## セキュリティ観点

### 確認すべきポイント

```bash
$ ls -la ~/.ssh/
# id_rsa は 600 か？
# .ssh ディレクトリは 700 か？
```

```bash
$ ls -la /etc/shadow
# root以外が読めないか？（通常は 640 または 600）
```

### 隠しファイルの発見

```bash
$ ls -la /tmp/
# 攻撃者が隠したファイル（.xxx）がないか
```
